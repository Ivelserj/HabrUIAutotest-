TEST PLAN FOR HABR.COM WEBPAGE

This test plan contains UI test cases designed for automated UI testing. Each test case is atomic and independent, with no overlapping steps between cases.

  • Locate and verify the "Авторы" (Authors) tab is present and visible
  • Locate and verify the "Компании" (Companies) tab is present and visible
4. Verify other significant main page elements:
  • Verify the "Хабр" logo link is present and visible
  • Verify the "Все потоки" (All Streams) link is present and visible
  • Verify the "Поиск" (Search) link is present and visible
  • Verify the "Написать публикацию" (Write Publication) link is present and visible
  • Verify the "Настройки" (Settings) button is present and visible
  • Verify the "Войти" (Login) button is present and visible
  • Verify the main content area is present and visible
  • Verify the footer section is present and visible
EXPECTED RESULTS
  • All main page elements listed above are present on the page
  • All elements are visible and displayed correctly
  • Header element with class "tm-header__container" is found and displayed
  • All content tabs ("Статьи", "Посты", "Новости", "Хабы", "Авторы", "Компании") are present and visible
  • All other significant main page elements are present and visible
